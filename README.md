# Omni Notion

Inject some CSS to custom Notion colors ;)

![Preview](.github/preview.png)

---

## 4 Steps to Use it.

1. Install

```sh
npm i
```

2. Build

```sh
npm run build
```

3. Run your new app.
   _It will be at dir ./packages_

4. Put your Notion in Dark Mode ;)
