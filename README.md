# Notion Omni

Inject some JS/CSS to customize Notion ;)

![Preview](.github/preview.png)

---

## 4 Steps to use it.

1. Install

```sh
yarn
```

2. Build

```sh
yarn package
```

3. Run your new app.
   _It will be at dir ./out_

4. Put your Notion in Dark Mode ;)
